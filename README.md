# Random-fact-app
Random Facts App

Description
The  "Random Facts App" is a simple web-based project designed to generate and display random facts. Each time the user clicks the "Show Another Fact" button, a new fact is displayed on the screen. The purpose of this activity is to demonstrate the integration of HTML, CSS, and JavaScript in creating an interactive and responsive mini web application.

Features
• Displays random facts dynamically using JavaScript.
• Clean and responsive design using CSS.
• Fully functional on both mobile and desktop browsers.
• Can be used offline without the need for an internet connection.

Technologies Used
• HTML5 – for webpage structure.  
• CSS3 – for layout and visual styling.  
•JavaScript – for interactivity and logic.
